# kbd106us-mac

## Setup

1. `JISUS.keylayout` を `~/Library/Keyboard Layouts` にコピー
2. 一度再起動 or ログアウト/ログイン
3. システム環境設定 → キーボード → 入力ソース
    * 英字のレイアウト: `JISUS`
    * "¥"キーで入力する文字: `\ (バックスラッシュ)`
4. `MyRemapKey.sh` を適当な場所に配置し、Automatorでアクション化、スタートアップ項目に登録
